## 使用方法

```bash
# 安装依赖
pnpm install

# 运行一次，生成配置文件
pnpm run start
```

然后把`ablacklist.sql`导入数据库，修改`config.json`中的配置，再运行一次。

## 构建

```bash
pnpm run build
```
